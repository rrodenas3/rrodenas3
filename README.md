## Working principles

- Harness before autonomy.
- Tool use must be permissioned, observable, and revocable.
- Financial and destructive actions need human approval boundaries.
- Agent demos are easy; production agents need identity, audit, evaluation, cost control, and operating ownership.
- The strongest AI products sell trusted work, not model access.
